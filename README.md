# Blacksmith Clicker

This is a simple CLI game where the user upgrades a sword from level 1 to level 100. The user wins this game if he/she gets Lv.100 sword.
Please refer to [99.term-project-proposal](./99.term-project-proposal.pdf) for detailed specifications.

## Prerequisites

This game is built with F#.
Please ensure that `dotnet 10.0` is installed on your local machine.

## How to build this project?

You can compile the project by typing `dotnet build`.

## How to run this project?

You can run this project by typing `dotnet run`.
